Serveur de Voeux



Bienvenue sur le serveur de voeux

Le serveur de voeux est un outil permettant aux membres du département informatique de centraliser toutes les demandes des enseignements pour lesquelles ils veulent participer. Cet outil permet en autre :

  • De centraliser les demandes des enseignements
  • De centraliser les heures disponibles pour les enseignants
  • De visualiser automatiquement
    • Les enseignements non couverts
    • Les conflits sur des demandes faites par différents enseignants pour un même enseignement
  • De permettre l'état de prévisionnelle des filières
  • De permettre l'état définitif des filières
